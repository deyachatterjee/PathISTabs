HELP

This program will parse text files created from PDF dumps obtained from CoPath Natural Language Search IIa (no race). For additional information contact germancz81@gmail.com.

INSTRUCTIONS

1. Copy the contents of the PDF file obtained from a Copath NLS II (no race) search and save the text file (*.txt) in a directory of your preference.

2. Fill in the options: text file path, output project name, site and specimen class.

3. Run.

4. The project will be saved in the "job_output" folder wich will open automatically once the job is done.

OUTPUT

1. Tab-delimited text file (/TABDELIM_CASES_OUT_x (x=time serial).txt)
SURG_NUM -> ACCESS_DATE -> SIGN_DATE -> SEX -> AGE -> NAME -> MRN_NUM -> MRN -> DIAGNOSIS -> FORMULA (0|1 if keyword present in DIAGNOSIS field)

2. JSON formated file with SURG_NUMs as keys (/JSON_CASES_x (x=time serial).txt)
[{SURG_NUM: [ACCESS_DATE, SIGN_DATE, SEX, AGE, NAME, MRN_NUM, MRN, DIAGNOSIS],...]],...}]

3. JSON formated file with MRNs as keys (/JSON_MRN_x (x=time serial).txt)
[{MRN, CASES: [{SURG_NUM, ACCESS_DATE, SIGN_DATE, SEX, AGE, DIAGNOSIS}, {}], NAMES: [NAME1, NAME2,], ALIAS_MRNS: [MRN1, MRN2,] }, {}]
